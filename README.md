# Flask Deployment for Grand-Bassam Page

This folder contains a simple Flask app to serve `index.html` and static assets.

## Setup

1. Create a virtual environment:

```bash
python -m venv venv
```

2. Activate it:

```bash
# Windows
venv\Scripts\activate
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the app:

```bash
python app.py
```

5. Open in browser:

```bash
http://127.0.0.1:5000/
```
